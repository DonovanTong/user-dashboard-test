 User Dashboard Trial#

A simple user dashboard built with **Next.js** and **Tailwind CSS**

## Getting Started

1. Install dependencies:
    ```bash
    npm install
2. Run dev server:
    ```bash
    npm run dev
3. Open http://localhost:3000

## Features
- Dashboard list view
- Live search by name
- Detail page with edit form

## Tech Stack
- Next.js (React + Node.js)
- Tailwind CSS
- Typescript
